# 🎬 AI Movie Insight Builder

A full-stack web application that allows users to enter an IMDb movie ID and get detailed movie insights including metadata, cast, and AI-powered audience sentiment analysis.

## 🚀 Live Demo
https://ai-movie-insight-builderr-blond.vercel.app/
## 📂 GitHub Repository
https://github.com/Tejaskmohare/ai-movie-insight-builder

---

# 📌 Features

- Enter an IMDb movie ID (example: tt0133093)
- Fetch movie details including:
  - Movie title
  - Poster
  - Cast list
  - Release year
  - IMDb rating
- Display short movie plot
- Responsive UI for desktop and mobile
- Input validation and error handling

📊 How It Works

User enters an IMDb movie ID.
The system fetches movie metadata.
Audience reviews are analyzed.
The app displays insights.

⚠️ Assumptions

IMDb ID must be valid.
Review availability may vary depending on the movie.
